Integração API sDoc

Como integrar sua aplicação com a API de assinaturas do sDoc

Versão
1.0
Atualizado em
30/03/2026
Responsável
Lucas Granado Gomes
Este manual descreve o fluxo de integração com a API de assinaturas do sDoc, cobrindo autenticação, envio de arquivos, criação de solicitações e posicionamento de carimbos.

Pré-requisito: Obter o Token de Acesso

Para iniciar a integração, é necessário um token de API vinculado à sua conta. O token pode ser obtido:

1.Por um administrador via portal sDoc Admin (usuários White Label).
2.Mediante contratação da API para clientes sDoc.

OBS: Cada conta tem direito a um token único. O token não expira, mas pode ser renovado quando necessário.

Fluxo de Integração

O fluxo da API segue três etapas principais:

1Enviar o Arquivo

Faça uma requisição de upload do arquivo. Um identificador temporário (`tempIdentifier`) será retornado. Guarde-o para usar na próxima etapa.

2Criar a Solicitação

Com o `tempIdentifier`, crie a solicitação informando no corpo da requisição:

1.`"title"` – Título da solicitação
2.`"description"` – Descrição
3.`"callback"` – Endereço de callback
4.`"organization"` com `"identifier"` e `"folder"` – identificadores da organização e pasta
5.`"settings"` – configurações de envio por e-mail, WhatsApp, SMS
6.`"files"` – dados do arquivo com o `tempIdentifier`
7.`"participants"` – dados dos participantes (nome, e-mail, CPF, telefone, tipo de assinatura)

Para obter o identificador da organização, utilize o endpoint:

GET https://sdocs.safeweb.com.br/api/integration/v1/organizations

Para obter o identificador das pastas dentro de uma organização:

GET https://sdocs.safeweb.com.br/api/integration/v1/organization/{IdentificadorOrg}/folders

3Posicionar o Carimbo do Participante

Caso o tipo de assinatura seja eletrônica, o posicionamento do carimbo é obrigatório. Use o endpoint:

POST https://sdocs.safeweb.com.br/api/integration/v1/participant/{ParticipanteID}/stamp

No corpo da requisição, informe:

1.Página onde o carimbo será aplicado
2.Coordenadas X e Y em `"position"`
3.Largura e altura da imagem em `"size"`

OBS: Após a solicitação ser criada, o envio é realizado automaticamente pelos canais de comunicação configurados (e-mail, WhatsApp, SMS).
